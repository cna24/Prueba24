LENGUAJE DE PROGRAMACIÓN C

VOL 1.0 - KERNIGHAN & RITCHIE STYLE

01. INTRODUCCIÓN ::

C es un lenguaje de programación de propósito general originalmente desarrollado por Dennis Ritchie entre 1969 y 1972 en los Laboratorios Bell.

Es conocido por su eficiencia y es el lenguaje base de muchos sistemas operativos modernos, incluyendo UNIX, Linux y Windows. C proporciona acceso de bajo nivel a la memoria mediante el uso de punteros, lo que lo hace ideal para la programación de sistemas.

02. ESTRUCTURA BÁSICA ::

Todo programa en C debe tener una función principal llamada main. Desde aquí comienza la ejecución.

/* hello_world.c */ #include <stdio.h> // Función principal int main() { // Imprime en consola printf("Hola, Mundo Digital.\n"); // Retorna éxito al SO return 0; }

03. CARACTERÍSTICAS ::

  • SIMPLICIDAD: Un conjunto reducido de palabras clave.
  • PORTABILIDAD: El código puede compilarse en diversas arquitecturas.
  • PUNTEROS: Acceso directo a direcciones de memoria.
  • VELOCIDAD: Genera código máquina altamente optimizado.

04. GESTIÓN DE MEMORIA ::

A diferencia de lenguajes modernos con Garbage Collectors, en C el programador es responsable de la memoria.

int *ptr; ptr = (int*) malloc(sizeof(int)); // Asignar memoria if (ptr == NULL) { printf("Error de memoria.\n"); } else { *ptr = 42; free(ptr); // Liberar memoria (CRÍTICO) }
